# Customer Churn Analysis Dashboard (Tableau + SQL)

## 📊 Project Overview

This project analyzes customer churn behavior using SQL and Tableau.
The dashboard helps identify churn patterns based on:

* Contract Type
* Internet Service
* Payment Method

The goal is to understand why customers leave and support business decision-making.

---

## 🛠 Tools Used

* SQL (Data Analysis)
* Tableau (Dashboard Visualization)
* Excel / CSV (Data Source)

---

## 📂 Dataset Files

* contract_churn.csv
* internet_churn.csv
* payment_churn.csv

---

## 📈 Key Insights

* Month-to-month contracts show higher churn rates
* Fiber optic internet users have higher churn
* Electronic check payments have higher churn probability

---

## 🧾 SQL Analysis

The SQL file contains queries used to:

* Join multiple churn tables
* Calculate churn rate
* Analyze churn by contract and payment type

File: `churn_analysis.sql`
